Algo Analyst

A Sorting Algorithm Analysis Web Application

About

This project aims to analyse the sorting algorithm to be specific. Where user have option to write his/her algorithm in any one of the language (C, Java, Python, Python3) and completes the program. The template provided by application accepts the input and pass the arr as parameter to the function which is to be completed by user.

The user written code is sent to the server where it is written onto a file and respective compiler is triggered to execute the program about 5 times with different set and number of input and the execution time is logged every time. Then all the data is sent back to client where a String parser parse the response and creates an array of the execution time

Technologies used in developing this project are:

Front End:
1. HTML
2. CSS
3. Javascript

Back End:
1. PHP
2. Javascript
